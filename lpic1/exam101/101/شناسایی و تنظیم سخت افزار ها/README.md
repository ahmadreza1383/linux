### حذف و نصب یک ماژول 
  


### حذف ماژول

برای حذف یک ماژول میتونید از دستور زیر استفاده کنید 


```bash 
rmmod {name}
```

 استفاده کنید lsmod برای دیدن ماژول ها از دستور  

________________

### نصب ماژول 


برای نصب ماژول از دستور زیر استفاده کنید 


```bash 
modprobe {name}
```
 
استفاده کنید lsmod برای دیدن ماژول ها از دستور
